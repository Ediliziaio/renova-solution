# Renova Solution

Sito web aziendale per Renova Solution - Azienda specializzata in fotovoltaico, infissi, ristrutturazioni e climatizzazione in Friuli Venezia Giulia e Veneto.

## 🚀 Tecnologie

- React 19 + TypeScript
- Vite
- Tailwind CSS
- React Router DOM
- Framer Motion
- shadcn/ui

## 📦 Installazione

```bash
npm install
```

## 🏃‍♂️ Sviluppo

```bash
npm run dev
```

## 🔨 Build

```bash
npm run build
```

## 🌐 Deploy

Il sito è configurato per il deploy su Vercel.
